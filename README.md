# Доктор Прыгун

**Доктор Прыгун** – это аркадная игра на Python с использованием библиотеки Tkinter.  
Вы управляете доктором, который прыгает по платформам, собирает лекарства и избегает препятствий.  
Игра бесконечна – чем выше вы забираетесь, тем сложнее становится путь.

## Управление
- **←** – движение влево  
- **→** – движение вправо  
- **R** – перезапуск игры после поражения

## Цель
Набрать как можно больше очков (лекарств), не упав вниз и не столкнувшись с красными препятствиями.

## Требования
- Python 3.x
- Tkinter (входит в стандартную поставку Python)


## Особенности
- Генерация платформ, звёзд (лекарств) и препятствий происходит автоматически.
- При сборе звезды счёт увеличивается на 1.
- При столкновении с препятствием или падении игра заканчивается – можно нажать **R** для новой попытки.
- Экран автоматически прокручивается вверх, когда персонаж достигает верхней половины окна.
